# img2bin (960x220 version)

A user-friendly image grid editor that allows users to load images, apply adaptive thresholding, and export the grid as hexadecimal values. The tool is specifically designed for handling grids of size 96x22 with support for customization options like block size, constant value, and opacity.

---

## Features

- Load images and visualize them on a 96x22 grid.
- Apply adaptive thresholding with adjustable block size and constant value.
- Edit the grid interactively with mouse clicks.
- Export the grid data as hexadecimal values.
- Clear the canvas with a single button.

---

## Requirements

Before using this software, ensure you have the following installed:

- **Qt 5** (or higher)
- **OpenCV** (4.x recommended)
- A C++17-compatible compiler (e.g., GCC, Clang, MSVC)
